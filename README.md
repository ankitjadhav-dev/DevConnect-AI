# DevConnect AI 🚀

DevConnect AI is a MERN Stack developer networking platform with AI-powered features using Google's Gemini API.

## Features

- User Authentication (Login/Register)
- Protected Routes
- Create, Edit and Delete Posts
- Like and Comment System
- Profile Management
- AI Post Generator
- AI Developer Assistant
- Responsive Design
- Dark/Light Theme

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT (JSON Web Token)

### AI Integration
- Google Gemini API

## Project Structure

client/
- React Frontend

server/
- Express Backend
- MongoDB Models
- API Routes
